Changelog

1.2.4.6 (2011-06-14)

  • Fixed an issue with GAFYD account management on the options page (they can now be deleted properly again)

1.2.4.5 (2011-04-30)

  • Fixed an issue with browser action icon rendering and animation

1.2.4.4 (2011-04-26)

  • Added a loading animation on the popup page when the extension is waiting for an outgoing request
  • Client storage backend rewritten (bye LocalStorage, hello IndexedDB).
    You will have to re-enter any custom settings, sorry!
  • New icon set, OSX (thanks to Marijn van der Werf)

1.2.4.2 (2011-04-21)

  • Added experimental one-way label support

1.2.4.1 (2011-04-16)

  • Using actions in the preview window will show the next (or previous) unread mail instead of returning to compact mode
  • Update alerts regarding the extension will be displayed in a notification instead of in a new tab

1.2.4.0 (2011-04-14)

  • Added previous- and next-links in expanded preview window to easily navigate between unread mail
